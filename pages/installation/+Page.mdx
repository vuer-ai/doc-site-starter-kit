## Install with your package manager

All three major Node.js package managers are supported. We recommend **pnpm** for
its superior speed and disk efficiency, especially in monorepo setups.

### <span className="px-2 py-0.5 rounded bg-indigo-100 text-indigo-700 text-xs font-bold">RECOMMENDED</span> pnpm

```bash
pnpm install
```

### npm

```bash
npm install
```

### yarn

```bash
yarn install
```

## Adding to an existing project

If you want to integrate the stack into an existing Vite project rather than cloning
the starter kit, install the dependencies individually:

### Runtime dependencies

```bash
pnpm add react react-dom vike vike-react
```

### Dev dependencies

```bash
pnpm add -D @types/react @types/react-dom @vitejs/plugin-react typescript vite @tailwindcss/vite tailwindcss
```

## Verify the installation

After installing, run the development server to confirm everything is working:

```bash
pnpm dev
```

You should see output similar to:

```text
  VITE v6.x.x  ready in 312 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

## Troubleshooting

<div className="space-y-4">

<Callout type="warning">
**"Cannot find module 'vike/plugin'"**

This usually means the vike package is not installed or your node\_modules is corrupted.

<CodeBlock language="bash" code="rm -rf node_modules pnpm-lock.yaml && pnpm install" />
</Callout>

<Callout type="info">
**Tailwind styles not applying**

Ensure the `@tailwindcss/vite` plugin is listed before `@vitejs/plugin-react` in
your `vite.config.ts` plugins array. Also confirm your CSS entry file starts with `@import "tailwindcss"`.
</Callout>

<Callout type="info">
**TypeScript errors on React 19 types**

Make sure `@types/react` and `@types/react-dom` are both on version `^19.0.0`.
Run `pnpm update @types/react @types/react-dom` to upgrade.
</Callout>
